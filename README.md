## GitHub Pages 업데이트

1. 이 ZIP을 압축 해제합니다.
2. 압축을 푼 폴더 안의 파일과 폴더를 기존 `skin-reset-365` 저장소 폴더에 모두 복사합니다.
3. 기존 파일은 덮어씁니다.
4. GitHub Desktop에서 변경사항을 확인합니다.
5. Summary에 `v2.0.0 최종 통합 업데이트`를 입력합니다.
6. `Commit to main`을 누릅니다.
7. `Push origin`을 누릅니다.

## 주의

- 브라우저 데이터를 삭제하면 로컬 기록이 사라질 수 있습니다.
- 중요한 데이터는 정기적으로 `.skinreset` 파일로 백업하세요.
- 음식 정보는 참고용입니다. 제품·브랜드·조리법에 따라 원재료가 달라질 수 있으므로 실제 성분표를 확인하세요.
- 피부 분석은 의료적 진단이 아닙니다.
